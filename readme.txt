Enables shortcodes for embedded deployment of Caspio database applications.


== Description ==

Caspio is an online database featuring point-and-click tools for designing database tables, forms, reports, charts, and sophisticated business applications. Applications can be created without programming, and embedding them into a web page is as easy as copying and pasting a code snippet.

The Caspio Deployment Plugin allows you to embed Caspio apps seamlessly on any WordPress post or page, simply by pasting the provided shortcode. 

For information about Caspio, visit the official website at [http://www.caspio.com](http://www.caspio.com "Caspio Official Website").


== Installation ==

Install the WordPress plugin:

1. Download the Caspio Deployment Plugin and extract the files.

2. Copy the resulting directory into /wp-content/plugins/.

3. Activate the plugin through the 'Plugins' menu of WordPress.

Once the plugin is installed and activated, WordPress will accept the provided shortcodes when you deploy your Caspio DataPages. See the Screenshots tab for details.

= What are the requirements of Search Engine Optimized (SEO) deployment? =

To deploy a Caspio Report using SEO deployment, follow the steps below:

1. Install and activate the Caspio Deployment Plugin on your WordPress site.

2. Ensure your Caspio plan includes the SEO Deployment feature. Learn more at [http://www.caspio.com/pricing](http://www.caspio.com/pricing "Caspio Pricing").

3. Go to the deploy option of the Report DataPage configured for SEO. Click the WordPress icon and copy the shortcode provided in the "SEO" tab.

4. Paste the shortcode in the HTML source of your WordPress page or post.

To learn more about SEO deployment, read the help article at [http://howto.caspio.com/deployment/seo-deployment-directions/](http://howto.caspio.com/deployment/seo-deployment-directions/ "Caspio SEO Deployment Directions").
